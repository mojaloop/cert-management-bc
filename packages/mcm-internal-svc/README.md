# Internal MCM (Mojaloop Certificate Management) SVC
This internal API service is responsible for managing the public certificates.

Default port: 3200

## Build

```bash
npm run build
```

## Run this service

Anywhere in the repo structure:
```bash
npm run start:dev
```

## Auto build (watch)

```bash
npm run watch
```

## Unit Tests

```bash
npm run test:unit
```

## Integration Tests

```bash
npm run test:integration
```
